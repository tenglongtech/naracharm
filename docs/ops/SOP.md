> 涵盖订单履约、物流、客服、退换货、内容日历。
> 国内制作 + 国际直邮模式。上线后据此执行。

## 一、订单履约（国内制作 + 国际直邮）

### 1.1 物流方案对比

| 方案 | 时效 | 成本 | 适合 |
|---|---|---|---|
| **4PX 专线 / 云途** | 7–15 天 | 低（按克） | ⭐ 常规订单主选，性价比最高 |
| **ePacket（邮政）** | 10–20 天 | 低 | 备选，部分国家不稳 |
| **邮政 EMS** | 7–12 天 | 中 | 稳但稍慢 |
| **DHL / FedEx** | 3–5 天 | 高 | 高客单加急付费订单 |

**主策略**：
- 常规订单 → **4PX 专线**（7–15 天，含追踪）
- **满 $120 免运费**（提升客单价，与设计稿全站口径一致）
- 高客单（$150+）可选 **加急 $15–25 升级 DHL**
- 首饰轻（< 100g），运费按件/克计，成本可控

### 1.2 订单处理流程

```
新订单到达（Stripe webhook → status=paid）
  ↓
后台收到通知（Payload Admin / 邮件提醒）
  ↓
确认库存 + 打印面单（4PX 后台/插件）
  ↓
手工制作/质检/包装（含品牌卡片 + 感谢信）
  ↓
发货 → 录入追踪号 → 自动触发"发货邮件"
  ↓
追踪号同步到订单 → 顾客可查物流
  ↓
妥投后 7 天 → 发"评价邀请"邮件
```

**时效承诺**：
- 制作 + 发货：付款后 **1–3 个工作日**发出（手工品如实说明）
- 运输：**7–15 个工作日**到欧美
- 站点显著位置标注（FAQ + 商品页 + 结账页），避免售后纠纷

### 1.3 包装（提升开箱体验 = 复购与社媒传播）

- 品牌首饰盒 / 绒布袋（印 logo）
- 品牌卡片（手写感谢 + 工艺故事简介）
- 保养说明卡
- 防氧化密封袋
- 外箱朴素（防盗 + 海关），内里精致

**目标**：让顾客愿意拍开箱视频发 TikTok——免费的 UGC 内容。

## 二、客服

### 2.1 渠道
- **主**：邮箱 support@[品牌名].com（24h 内回复）
- **辅**：站内聊天 Tidio（工作时间在线）
- **社媒**：TikTok/Instagram DM（定期查）

### 2.2 FAQ 标准答案（建站时落页）

| 问题 | 标准回答要点 |
|---|---|
| 多久到货？ | 制作 1–3 工作日 + 国际运输 7–15 工作日 |
| 运费？ | 满 $120 免运费；否则按地区 $4.9–8.9 |
| 关税？ | 美国 $800 以下免关税；欧洲可能收 VAT（买家承担） |
| 材质安全？ | 925 银/黄铜，低敏；注明含铅含镍情况 |
| 能退换吗？ | 30 天无理由，详见退换货政策 |
| 尺寸？ | 商品页标注长度；项链 16/18/20 英寸可选 |
| 定制？ | 暂不支持/支持（按业务决定） |
| 物流查询？ | 提供追踪号 + 查询链接 |

### 2.3 客服原则
- **快**：24h 内首响
- **暖**：手工品牌的人格化语气，不机械
- **透明**：物流慢就主动告知 + 小补偿（如下单 5% 优惠券）

## 三、退换货政策

### 3.1 政策（落页 + 结账页可见）

- **30 天无理由退货**：收到货 30 天内可申请，商品未佩戴/未损坏
- **退货地址**：香港仓（降低国际退货成本，顾客承担退回运费）
- **退款**：收到退货 5 工作日内原路退款（不含原运费）
- **质量问题**：免费补发或全额退款（含运费），无需退货
- **定制款**：不支持无理由退货（若开展定制业务）

### 3.2 处理流程
```
顾客申请（邮件/表单）
  ↓
客服确认原因 + 给香港退货地址
  ↓
顾客寄回（承担运费）
  ↓
香港仓收货验收
  ↓
退款 / 补发
```
