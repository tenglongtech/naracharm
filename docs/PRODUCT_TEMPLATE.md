# 产品录入模板

> 请按此模板填首批 8–15 个 SKU。填好后放在 `assets/products/catalog.md`（或我给 CSV）。
> **你提供：产品图 + 名称 + 描述**，价格/库存可先留空让我按区间建议。

---

> ⭐ **每件首饰必须有一个故事** —— 这是品牌灵魂。
> 故事写法见 [`docs/brand/STORY_DOCTRINE.md`](brand/STORY_DOCTRINE.md)（四层结构：来路 / 手艺 / 寓意 / 共鸣）。
> **没有故事的产品不发。** 你给故事素材，我润色；或你只给文化来源+材质，我按四层代写。

---

|---|---|---|

## 五、图片拍摄要点（你已有产品图，对照检查）

- [ ] 至少 1 张**白底/浅底**主图（用于列表页统一感）
- [ ] 至少 1 张**佩戴图**（颈间/手腕/耳部/手机挂），帮顾客想象
- [ ] 至少 1 张**细节特写**（纹理、材质、工艺）
- [ ] 自然光拍摄，避免闪光灯硬光
- [ ] 分辨率 ≥ 1200×1200（详情页放大用）
- [ ] 命名规范：`品类-款式-编号.jpg`（如 `necklace-lotus-01.jpg`）

详细拍摄 SOP 见 `docs/ops/PHOTOGRAPHY.md`（Phase 1 产出）。
