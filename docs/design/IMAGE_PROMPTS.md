## 🎨 通用风格说明（每次生成前先发这段给 ChatGPT）

```
我正在为手工首饰品牌 Nara Charm 生成图片。请所有图片严格遵循这套视觉风格：

【调性】温暖、复古、手工质感、文艺克制。不要廉价、不要华丽浮夸、不要商业硬广感。
【配色】
- 主背景：奶白米色 #FAF7F1（温暖不刺眼的白）
- 点缀色：砖红 #C4745A、深棕 #3A2E2B、金色 #D4AF37
- 可用辅助色：淡紫(紫水晶)、淡粉(粉水晶)、青蓝(绿松石)、黑(黑曜石)
【光线】自然柔和的侧光/窗光，不要硬光和闪光灯。营造手工工坊的温暖感。
【质感】必须能看到材质纹理：水晶的透光、绳的编织纹理、银珠的微光/氧化。
【构图】留白充足、克制，不要塞满。欧洲精品店的克制高级感。
【产品】品牌主打天然水晶圆珠手链（8mm，配苗银隔珠）和民族风手机挂链（编绳+宝石珠）。
【禁止】不要任何文字、水印、logo、可读字样。

请记住这套风格，我接下来会逐张描述具体画面。
```

## 💡 生成贴士

1. **顺序**：按优先级先生成 1-2 张（Hero + 包装），这两张对首页视觉影响最大。
2. **一张一张生成**：GPT-4o 单张质量最高，不要一次让它生成多张。
3. **不满意就追问**：
   - 太亮 → "再暗一点，更工坊氛围"
   - 太满 → "留白再多一些"
   - 质感不够 → "水晶透光要更明显，能看到内部纹理"
   - 颜色偏了 → "背景再暖一些 / 更奶白一些"
4. **命名**：按表格里的文件名存，放到 `设计稿/` 目录，我批量接入。
5. **风格一致性**：每次生成都先确认它记得"通用风格说明"，中途如果风格漂移，重新发一遍那段说明。

## ⚠️ 关于产品主图的再次提醒

这份 prompt 清单**不包含产品主图**（商品页大图、列表缩略图）——那些必须用你真实产品的照片，因为：
- AI 生成的产品图和你实际做出来的产品对不上 → 退货/差评风险
- 你已经有 21 张产品图接入站点了，足够

如果某款产品**还没拍**，想先用 AI 概念图占位也可以，但上线前务必换成真图。
