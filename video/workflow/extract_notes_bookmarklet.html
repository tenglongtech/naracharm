🍇 格蕾丝Grace 笔记提取

📋 操作步骤

  1. 在浏览器打开格蕾丝的主页：
    xiaohongshu.com/user/profile/66ec01d00000000009019c22
  2. 等页面加载完成，慢慢滚动让所有笔记都加载出来
  3. 按 F12 → 点 Console (控制台)
  4. 粘贴下面的代码 → 按 Enter
  5. 复制输出的所有链接，发给我

📝 复制以下代码：

javascript:(async function(){
  const ids = new Set();

  // 从所有链接提取笔记ID
  document.querySelectorAll('a[href*="explore/"]').forEach(a => {
    const m = a.href.match(/explore\/([a-f0-9]{24})/);
    if (m) ids.add(m[1]);
  });

  // 尝试调用API获取更多（利用你已登录的cookie）
  try {
    const r = await fetch(
      'https://edith.xiaohongshu.com/api/sns/web/v1/user/notes?' +
      new URLSearchParams({ user_id: '66ec01d00000000009019c22', num: '30', cursor: '' }),
      { credentials: 'include' }
    );
    const data = await r.json();
    if (data.data?.items) {
      data.data.items.forEach(i => {
        if (i.note_card?.note_id) ids.add(i.note_card.note_id);
      });
    }
  } catch(e) {}

  // 如果上面不行，尝试另一个API
  if (ids.size < 5) {
    try {
      const r2 = await fetch(
        'https://edith.xiaohongshu.com/api/sns/web/v1/user_posted?' +
        new URLSearchParams({ user_id: '66ec01d00000000009019c22', count: '30' }),
        { credentials: 'include' }
      );
      const data2 = await r2.json();
      console.log('API2:', data2);
    } catch(e) {}
  }

  // 输出结果
  const urls = Array.from(ids).map(id => 'https://www.xiaohongshu.com/discovery/item/' + id);
  console.log('✅ 找到 ' + urls.length + ' 个笔记:\n');
  urls.forEach((u, i) => console.log((i+1) + '. ' + u));
  console.log('\n📋 复制下面这行全部内容发给我:\n---');
  console.log(urls.join('\n'));
  console.log('---');

  // 自动复制到剪贴板
  const text = urls.join('\n');
  try {
    await navigator.clipboard.writeText(text);
    console.log('✅ 已自动复制到剪贴板!');
  } catch(e) {
    console.log('⚠️ 请手动复制上面的链接');
  }
  return urls;
})();

⚠️ 如果遇到跨域问题

直接复制下面的代码贴到 Console 运行（不要用 bookmarklet 方式）：

// 直接复制这段到 Console 按回车
